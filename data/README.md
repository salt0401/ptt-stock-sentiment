# Data Directory

## Layout

```
data/
├── comments/
│   └── raw/                          # Scraped PTT data
│       ├── after_market.pkl          # After-market discussion posts (auto-generated)
│       └── during_market.pkl         # During-market discussion posts (auto-generated)
├── financial/
│   ├── *.csv                         # Monthly stock price files (2018-1.csv ... 2023-12.csv)
│   ├── market_daily_price.csv        # Daily market price data
│   ├── daily_returns_five_factor.csv # Daily returns for five-factor model
│   ├── ROE.csv                       # Return on equity data
│   ├── total_assets.csv              # Total assets for growth rate calculation
│   ├── aggregated_table/             # Aggregated data tables
│   ├── excel_raw/                    # Original Excel source files from TEJ
│   ├── fama-french/                  # Fama-French model outputs and reference data
│   └── prediction_features/          # Additional features for prediction models
│       ├── stock_indices/            # Stock market indices
│       ├── pe_ratios/                # Price-to-earnings ratios
│       ├── us_indices/               # US stock market indices (DJI, S&P500, NASDAQ, SOX)
│       └── corporate_governance/     # Corporate governance data
├── nlp_models/                       # CKIPTagger model files (manual setup required)
│   ├── embedding_character/
│   ├── embedding_word/
│   ├── model_ws/
│   ├── model_pos/
│   └── model_ner/
└── external/
    └── csenti_package/               # Chinese sentiment lexicons (ANTUSD, Opinion Treebank)
```

## Required Input Files

These files must exist before running pipelines:

| File | Used By | Description |
|---|---|---|
| `financial/{year}-{month}.csv` | `run_five_factor.py` | Monthly stock price data (2018-2023) |
| `financial/ROE.csv` | `run_five_factor.py` | Quarterly ROE data per company |
| `financial/total_assets.csv` | `run_five_factor.py` | Quarterly total assets per company |
| `nlp_models/*` | CKIPTagger sentiment | Pre-trained NLP model files |

## Auto-Generated Files

These are created by the pipelines and can be regenerated:

| File | Generated By |
|---|---|
| `comments/raw/after_market.pkl` | `run_scraper.py` |
| `comments/raw/during_market.pkl` | `run_scraper.py` |
